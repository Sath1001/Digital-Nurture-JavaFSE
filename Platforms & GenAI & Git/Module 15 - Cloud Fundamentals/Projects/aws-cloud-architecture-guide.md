# Module 15: Cloud Fundamentals (AWS) Hands-On Guide

## Overview
This guide covers Cloud computing service models, AWS core infrastructure, networking, compute, storage, databases, and serverless architectures.

## 1. Cloud Service Models
- **IaaS**: AWS EC2, Elastic Block Store (EBS), Virtual Private Cloud (VPC).
- **PaaS**: AWS Elastic Beanstalk.
- **SaaS**: Managed software applications.

## 2. Core AWS Services
- **AWS EC2**: Virtual servers running Amazon Linux / Ubuntu.
- **AWS S3**: Object storage for asset files, static web hosting, and backups.
- **AWS RDS**: Managed MySQL / PostgreSQL DB instance setup in private subnets.
- **AWS Serverless**: AWS Lambda functions and AWS API Gateway HTTP routes.
