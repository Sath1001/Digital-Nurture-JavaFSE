# Module 16: Gen AI Fundamentals & GitHub Copilot Guide

## Overview
This guide covers Generative AI fundamentals, prompt engineering strategies, and practical usage of GitHub Copilot for Java Full Stack development.

## 1. Prompt Engineering Strategies
- **Zero-Shot Prompting**: Direct instructions without examples.
- **Few-Shot Prompting**: Providing input-output examples before the instruction.
- **Chain-of-Thought Prompting**: Instructing the AI model to reason step-by-step.

## 2. GitHub Copilot Integration
- Real-time inline code completions.
- Unit test generation with JUnit 5 & Mockito.
- Refactoring legacy code into Java Streams.
- Code review and security vulnerability scanning.
