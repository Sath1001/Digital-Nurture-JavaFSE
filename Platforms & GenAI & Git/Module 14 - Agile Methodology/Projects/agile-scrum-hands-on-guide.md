# Module 14: Agile Methodology & Scrum Framework

## Overview
This guide covers Agile principles, Scrum roles, ceremonies, artifacts, story point estimation, and burndown chart tracking.

## 1. Agile Manifesto (4 Core Values & 12 Principles)
- Individuals and interactions over processes and tools.
- Working software over comprehensive documentation.
- Customer collaboration over contract negotiation.
- Responding to change over following a plan.

## 2. Scrum Framework
- **Roles**: Product Owner (manages backlog), Scrum Master (facilitator), Development Team.
- **Ceremonies**:
  - Sprint Planning (Selecting user stories for sprint).
  - Daily Scrum (15-min standup meeting).
  - Sprint Review (Demonstrating completed increment).
  - Sprint Retrospective (Continuous improvement).
- **Artifacts**: Product Backlog, Sprint Backlog, Increment.

## 3. User Story Format & INVEST Principle
- **Format**: *As a [user role], I want [feature goal], so that [business benefit].*
- **INVEST Principle**: Independent, Negotiable, Valuable, Estimable, Small, Testable.
