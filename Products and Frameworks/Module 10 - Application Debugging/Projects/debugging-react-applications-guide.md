# Application Debugging Hands-On Guide

## Overview
This guide covers debugging techniques for React and Java applications using Chrome DevTools and Visual Studio Code.

## 1. Debugging in Visual Studio Code
- **Breakpoints**: Line breakpoints, Conditional breakpoints, and Logpoints.
- **Variables & Watch Window**: Inspecting local scope variables and adding watch expressions.
- **Stepping Controls**:
  - `F10` (Step Over)
  - `F11` (Step Into)
  - `Shift+F11` (Step Out)

## 2. Debugging React Applications with Chrome DevTools
- **React Developer Tools Extension**: Inspecting component hierarchy, props, state, and hooks.
- **Sources Panel**: Adding breakpoints in JSX/TypeScript files via Source Maps.
- **Console & Network Inspection**: Tracking API requests (Fetch/Axios) and payload responses.
